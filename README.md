# Developer Portfolio

개발자 **주현**의 이력과 프로젝트를 소개하기 위한 개인 포트폴리오 웹사이트입니다.

## 소개

이 저장소는 GitHub Pages를 통해 배포되는 개발자 포트폴리오를 관리합니다.
경력, 기술 스택, 주요 프로젝트와 연락처를 한곳에서 간결하게 보여주는 페이지로 만들어갈 예정입니다.

## 구성 예정

- 소개 및 한 줄 프로필
- 기술 스택
- 경력과 이력
- 주요 프로젝트
- GitHub 및 연락처
